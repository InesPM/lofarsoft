\subsection{\-Detailed \-Description}
\-Class object for stocking data parameters and write datas for the \-I\-C\-D6's \-Dynspec \-Group. 

\par
 \-Usage\-: \par
 \-This class need as parameter all metadata of the dynamic spectrum \-Goup for process the data. \-Data have 3 dimensions (at the opposite to \-I\-C\-D3 data which have 2). \-The additionnal dimension is \-Stokes. \-The function stock\-Dynspec\-Data stocks paramerter in private attributes, and the function write\-Dynspec\-Data process the data themselves.
