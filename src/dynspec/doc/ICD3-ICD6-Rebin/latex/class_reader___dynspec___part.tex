\subsection{\-Detailed \-Description}
\-Class object for read \-I\-C\-D3 files and put in variables metadatas for the \-I\-C\-D6's \-Dynspec \-Group. 

\par
 \-Usage\-: \par
 \-This class need as parameter a \doxyref{\-Stock\-\_\-\-Write\-\_\-\-Dynspec\-\_\-\-Metadata\-\_\-\-Part}{p.}{class_stock___write___dynspec___metadata___part} object. \-The read\-Dynspec function will read dynspec metadatas and stock them in the \doxyref{\-Stock\-\_\-\-Write\-\_\-\-Dynspec\-\_\-\-Metadata\-\_\-\-Part}{p.}{class_stock___write___dynspec___metadata___part} object as private attributes. \-These attributes will be written in the next step by stoc\-Dynspect\-Metadata function which is in the class \doxyref{\-Stock\-\_\-\-Write\-\_\-\-Dynspec\-\_\-\-Metadata\-\_\-\-Part.\-h}{p.}{_stock___write___dynspec___metadata___part_8h_source} \-In fact this class is very similar to \doxyref{\-Reader\-\_\-\-Root\-\_\-\-Part}{p.}{class_reader___root___part} class, the only difference is\-: this class is for dynspec metadata, so, the main code (\doxyref{\-Dynspec\-Part.\-cpp}{p.}{_dynspec_part_8cpp}) loop on dynspec to process and use this class several times (at the opposite of \doxyref{\-Reader\-\_\-\-Root\-\_\-\-Part}{p.}{class_reader___root___part} class which is called only once)
