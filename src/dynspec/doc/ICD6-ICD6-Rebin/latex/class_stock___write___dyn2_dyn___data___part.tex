\-Class object for stocking data parameters, processing the (select or rebin) and write data \-Matrix in the \-I\-C\-D6's \-Dynspec \-Groups. 

\par
 \-Usage\-: \par
 \-This class need as parameter all metadata of the dynamic spectrum \-Goup for process the data. \-Data have 3 dimensions. \-The additionnal dimension is \-Stokes. \-The function stock\-Dynspec\-Data stocks parameter (for rebinning the data) in private attributes, and the function write\-Dynspec\-Data process the data themselves.
