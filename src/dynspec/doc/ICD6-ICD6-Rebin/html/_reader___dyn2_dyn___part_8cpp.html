Overview:
This function readDynspec is described (programmed) here. It needs as parameter a Stock_Write_Dyn2Dyn_Metadata_Part object. The readDynspec function will read dynspec metadatas and stock them in the Stock_Write_Dyn2Dyn_Metadata_Part object as private attributes thanks to stockDynspecMetadata function.
