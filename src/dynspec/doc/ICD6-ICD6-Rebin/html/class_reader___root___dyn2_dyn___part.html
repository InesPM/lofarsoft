Detailed Description

Class object for read ICD3 files and put in variables metadatas for the ICD6's Root Group.


Usage:
This class need as parameter a Stock_Write_Root_Dyn2Dyn_Metadata_Part object. The readRoot function will read root metadatas and stock them in the Stock_Write_Root_Dyn2Dyn_Metadata_Part object as private attributes. These attributes will be written in the next step by stockRootMetadata function which is in the class Stock_Write_Root_Dyn2Dyn_Metadata_Part.h
